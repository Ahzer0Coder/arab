# 📚 منصة اختبار التذوق الأدبي

منصة تفاعلية لاختبار معرفتك في مادة التذوق الأدبي مع شروحات مفصلة وتقييم فوري.

## ✨ المزايا

- 🎯 **120 سؤال** موزع على 5 فصول
  - 65 سؤال اختيار من متعدد (MCQ)
  - 55 سؤال صواب وخطأ (True/False)

- 🎨 **تصميم احترافي**
  - واجهة عربية كاملة RTL
  - ألوان أسود وأحمر دموي
  - سريع الاستجابة على جميع الأجهزة

- 💡 **ميزات متقدمة**
  - شروحات مفصلة لكل سؤال
  - حساب فوري للنقاط
  - مراجعة شاملة للإجابات
  - تقدم بصري ديناميكي
  - 4 أنواع اختبارات مختلفة

## 📖 الفصول المتاحة

1. **أساسيات التذوق الأدبي** - المفاهيم والتعريفات
2. **أنواع التذوق** - السليم، السقيم، الإيجابي، السلبي
3. **العوامل المؤثرة** - البيئة، الزمان، التربية
4. **تحليل النصوص** - دراسة الشعراء والنصوص
5. **المراجعة النهائية** - أسئلة شاملة

## 🚀 كيفية الاستخدام

1. افتح ملف `index.html` في متصفح الويب
2. اختر نوع الاختبار المطلوب:
   - اختبار كامل
   - MCQ فقط
   - True/False فقط
   - اختبار حسب موضوع

3. أجب على الأسئلة بالنقر على الخيارات
4. استخدم زر "الشرح" لفهم الإجابة الصحيحة
5. بعد الانتهاء، شاهد النتائج والمراجعة

## 🎮 التحكم

- استخدم الأزرار للتنقل أو اضغط **أسهم لوحة المفاتيح** (← و →)
- يمكنك الرجوع لأي سؤال وتعديل إجابتك
- النقاط تُحسب فوراً

## 🎓 محتوى الأسئلة

جميع الأسئلة مأخوذة من:
- دليل مراجعة التذوق الأدبي (منتصف الفصل الثاني)
- المراجعة النهائية الشاملة
- أسئلة متنوعة MCQ و True/False

كل سؤال يحتوي على:
- ✅ النص الكامل للسؤال
- ✅ الخيارات (إن وجدت)
- ✅ الإجابة الصحيحة
- ✅ شرح مفصل للإجابة
